# Icons directory

Place your service icon files here. Supported formats: `.png`, `.svg`, `.jpg`, `.webp`

Reference them in `config.yml` like:

```yaml
- title: Grafana
  icon: grafana.png
  url: http://localhost:3000
```

If an icon file is missing, Dockboard will automatically try to load it from
[Simple Icons CDN](https://simpleicons.org) using the service title, and fall
back to a colored initial avatar if that also fails.

## Get icons from 
- https://dashboardicons.com/
- https://logosear.ch/
- https://www.vectorlogo.zone/
- https://www.svgrepo.com/vectors/
